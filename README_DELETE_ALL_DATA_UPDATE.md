# সব ডেটা মুছুন — আপডেট গাইড (Replace Guide)

## যা যা পরিবর্তন হয়েছে

**সমস্যা:** "সব ডেটা মুছুন" বাটনে ক্লিক করলে সরাসরি একটাই কনফার্মেশন ডায়ালগ আসত, টেক্সট প্লেইন এবং সাধারণ AlertDialog স্টাইলের ছিল। যে কেউ ভুলে দুইবার ক্লিক করলেই সব ডেটা মুছে যেত।

**সমাধান:** এখন ২-ধাপে ডেটা মুছবে —

1. **ধাপ ১ (সতর্কবার্তা):** নতুন, সুন্দর ডিজাইনের ডায়ালগ — উপরে লাল আইকন, বড় টাইটেল "সব ডেটা মুছে ফেলবেন?", ঠিক কী কী মুছে যাবে তার বিস্তারিত (ইনকাম, খরচ, হিসাব, সঞ্চয়, নোট) এবং নিচে লাল ওয়ার্নিং ব্যাজ।
2. **ধাপ ২ (পাসওয়ার্ড কনফার্ম):** "মুছে ফেলুন" চাপার পর এখন ৪-ডিজিট পাসওয়ার্ড বক্স আসবে। মেসেজ দেখাবে — *"আপনার অ্যাপের সম্পূর্ণ ডেটা মুছতে (0000) পাসওয়ার্ডটি ব্যবহার করুন"*।
   - প্রতিটা বক্সে একটা করে সংখ্যা টাইপ করলে অটোমেটিক পরের বক্সে কার্সর চলে যাবে।
   - Backspace দিলে আগের বক্সে ফিরে যাবে।
   - সঠিক পাসওয়ার্ড (`0000`) দিলে তবেই ডেটা মুছবে, ভুল দিলে "❌ ভুল পাসওয়ার্ড" মেসেজ দেখিয়ে বক্সগুলো খালি হয়ে যাবে (ডায়ালগ বন্ধ হবে না, ফের চেষ্টা করা যাবে)।

## পাসওয়ার্ড পরিবর্তন করতে চাইলে
`SettingsActivity.java` ফাইলে এই লাইনটা খুঁজুন:
```java
private static final String CLEAR_DATA_PASSWORD = "0000";
```
এখানে `"0000"` এর জায়গায় আপনার পছন্দের ৪-ডিজিট কোড বসিয়ে দিলেই হবে।

## নতুন/পরিবর্তিত ফাইল

| ফাইল | কাজ |
|---|---|
| `app/src/main/java/com/jrappspot/cashlipi/activities/SettingsActivity.java` | ডিলিট ফ্লো — ২-ধাপের ডায়ালগ লজিক |
| `app/src/main/res/layout/dialog_delete_all_warning.xml` | ধাপ ১ এর সতর্কবার্তা ডায়ালগ UI |
| `app/src/main/res/layout/dialog_delete_password.xml` | ধাপ ২ এর ৪-ডিজিট পাসওয়ার্ড ডায়ালগ UI |
| `app/src/main/res/drawable/bg_pin_box.xml` | পাসওয়ার্ড বক্সের বর্ডার (ফোকাস হলে লাল হয়) |
| `app/src/main/res/drawable/bg_danger_icon_circle.xml` | আইকনের পেছনে হালকা লাল বৃত্ত |
| `app/src/main/res/drawable/ic_trash_bin.xml` | ট্র্যাশ আইকন (ভেক্টর, টিন্টেবল) |

## যেভাবে বসাবেন
পুরনো প্রজেক্টের উপর এই ZIP এর `app/` ফোল্ডারটা replace করে দিন (উপরের টেবিলের ফাইলগুলো নতুন/পরিবর্তিত, বাকি সব আগের মতোই আছে)। এরপর Android Studio-তে Sync + Rebuild করলেই কাজ শেষ।
