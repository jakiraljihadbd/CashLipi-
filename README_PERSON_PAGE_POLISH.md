# দেনা-পাওনা ব্যক্তি পেজ — পলিশ আপডেট

## যা করা হলো

1. **হেডার** — "দেনা পাওনা" + "X জন যুক্ত আছে" দুই-লাইন ব্লক সরিয়ে একটাই চিকন লাইন
   বসানো হয়েছে, অ্যাপ বার কালারে (`@color/m3HomeBalanceContainer`) — টাইটেল
   "দেনা পাওনা ব্যক্তি ও প্রতিষ্ঠান" + ছোট করে মোট লেনদেন সংখ্যা।

2. **অপঠনযোগ্য পিংক চিপ বাদ** — কার্ডের "N টি লেনদেন" ব্যাজে আগে হালকা পিংক
   ব্যাকগ্রাউন্ড (#FCE7F0) + গাঢ় পিংক টেক্সট (#B3105A) ছিল, কনট্রাস্ট কম থাকায়
   পড়া কঠিন ছিল। এখন নিউট্রাল ইন্ডিগো-টিন্ট চিপে (`bg_txn_count_chip`) বসানো
   হয়েছে, টেক্সট কালার `colorPrimary`।

3. **সার্চ + ফিল্টার + থিম — এক লাইনে** — সার্চ বক্সের ঠিক পাশে দুটো আইকন বাটন
   যোগ হয়েছে:
   - ফিল্টার আইকন (বিদ্যমান `ic_filter` পুনর্ব্যবহার) — ট্যাপ করলে পপআপ মেনুতে
     সব/অপরিশোধিত/পরিশোধিত অপশন দেখায়। আগের হরাইজন্টাল চিপ-স্ক্রল বাদ দিয়ে
     জায়গা বাঁচানো হয়েছে। ফিল্টার সক্রিয় থাকলে আইকনের কোণে ছোট ডট দেখায়।
   - থিম-চেঞ্জ আইকন (নতুন `ic_theme_palette`) — ট্যাপ করলে ৫টা কার্ড-স্টাইলের
     একটা বেছে নেওয়ার ডায়ালগ দেখায় (ক্লাসিক/মিনিমাল/গ্র্যাডিয়েন্ট/বোল্ড/কমপ্যাক্ট),
     পছন্দ `SharedPreferences`-এ সেভ থাকে।

4. **ব্যক্তি কার্ড রিডিজাইন** — অ্যাভাটার গোল থেকে বড়, রাউন্ডেড-স্কয়ার
   (66dp, radius 18dp), নাম টেক্সট 18sp, কার্ড প্যাডিং বাড়ানো — আরও ফ্রেন্ডলি
   ও বড় দেখায়।

## টেস্ট করার আগে

- অন্তত ২-৩ জন Person + কিছু LedgerEntry (paid/unpaid মিশিয়ে) দিয়ে হেডারের
  লেনদেন সংখ্যা, ফিল্টার পপআপ, এবং থিম ডায়ালগ ঠিকমতো কাজ করছে কিনা যাচাই করুন।
- থিম বদলে দেখুন প্রতিটা স্টাইলে কার্ডের ব্যাকগ্রাউন্ড/এলিভেশন/প্যাডিং/অ্যাভাটার
  সাইজ ঠিকমতো বদলাচ্ছে কিনা।
- Android Studio/AIDE-তে একবার Build চালিয়ে unresolved resource নেই তা
  নিশ্চিত করুন।
